## 娱乐场所和服务(支持1.0.9)
<p>添加了两种建筑："风月场所"和仅可以修在城镇和城市的"高级娱乐场所"。这两个建筑分别生产"服务"和"奢靡服务"（请不要试图贸易这两种服务，因为需要的贸易容量非常大）。

<p>士兵、劳工和农民对服务有需求，而市民和贵族对奢靡服务有需求。

<p>此外，添加了在这两个建筑中工作的"艺人"，由农民或奴隶升级而来，这种人口也可能升级为市民。
